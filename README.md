# Interval

A focus timer with customizable Pomodoro sessions, circular progress ring, and Web Audio completion sounds.

## What it does

Interval is a Pomodoro-style focus timer with three modes (Focus, Short Break, Long Break) and full customization of durations and session counts. It features a circular SVG progress ring, Web Audio API completion sounds, browser notifications, and keyboard controls. Built for anyone who needs a distraction-free timer without accounts or cloud sync.

## Tech

- HTML / CSS / JavaScript (Vanilla)
- Satoshi font from Fontshare CDN
- Web Audio API for completion sounds
- SVG for circular progress ring

## Run locally

Download or clone the repo. Open index.html in a browser. No build step required.
